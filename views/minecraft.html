Minecraft Logic Circuits!

Gabriel Downs September 20, 2017

Hey, so as Prof. Siddiqi mentioned in class today, you can build all of the digital circuits we learned in class using redstone in minecraft! Below are some screenshots of simple gates and circuits I made.

The red dust acts as a wire, and the red torches can be used as negators. All the basic gates can be derived from these:

(Left to Right) A, -A, A or B, A and B


Note there is no inherent AND gate, but it is made from NOT's and OR's

The AND gate pictured is actually: -( ((-A) + (-B) )

Distributing the NOTs and ORs by de MOrgan's Laws shows this is equivalent to A*B.


From this we can derive a nice XOR gate:

Note again this uses a slightly more complicated structure than the XOR gate we described in class. Since AND gates are more difficult to construct this version tries to make use of ORs and NOTs.

The gate above can be given by: -( (A*B) + (-A) ) + -( (A*B) + (-B) )

So we onlyhave to use 1 AND gabe and, after distributing, we see this is equal to ( (-A) * B ) + ( (A) * (-B) )


We can use the XOR gate plus the AND to construct nice Half and Full Adders:

The input is a the bottom: (Left to Right) A, B

And output on top: (Left to Right) C, S


We can chain these together to form a full 2-bit Adder!

Again the Input is at the bottom: (Left to Right) A1, B1, A0, B0

And the output is at the top: (Left to Right) extra carry, S1, S0

We can see C1 (the first carry) rippling through from one adder to the next in the middle of the circuit.

Note the addition in the screenshot is: 01 + 01 = 10


A view of the Adder from the side shows the XOR gates chaining together:


Finally we can make an SR-Latch:

This also appears a bit different from the latch we described in class, but the behavior is the same.


Note much of this is not my own original work, you can find very elegant circuits people have designed at sites like https://minecraft.gamepedia.com/Tutorials/Basic_logic_gates


People have use this type of circuitry to make working computer components like the registers we talked about in class.

Combine with a few mods, some have even made fully function video games, like this incredible working pokemon clone.
